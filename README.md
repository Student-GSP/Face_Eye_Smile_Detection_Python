# Face_Eye_Smile_Detection_Python
Face, Eye & Smile Detection using OpenCV (Python)

## Project Overview
Developed a real-time face, eye, and smile detection system using Python and OpenCV. The application uses Haar Cascade Classifiers to detect human faces, eyes, and smiles from webcam video streams. Detected features are highlighted with bounding boxes in real time, demonstrating image processing and computer vision concepts. The project showcases skills in Python programming, OpenCV, object detection, and real-time image analysis.

## Features
- Real-time face detection
- Eye detection within detected faces
- Smile detection
- Live webcam processing
- Bounding box visualization

## Technologies Used
- Python
- OpenCV (cv2)
- Haar Cascade Classifiers

 ## Applications 
- Facial expression analysis
- Attendance systems
- Human-computer interaction
- Computer vision learning projects
